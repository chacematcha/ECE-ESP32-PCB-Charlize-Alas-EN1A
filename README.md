# NPK SENSOR IDIYANALE

This project is an NPK Sensor System made using an Arduino Uno. This project was created for our Physics subject to help measure the nutrients present in the soil. The system can detect the amount of Nitrogen (N), Phosphorus (P), and Potassium (K) in the soil and display the results on the LCD screen. These three nutrients are important because they help plants grow healthy and strong.

The coding is used to control the whole system. The Arduino Uno reads the values from the NPK sensor and sends the data to the LCD display. The program also checks if the soil is healthy or unhealthy based on the nutrient values detected. If the values are within the correct range, the LCD will show “HEALTHY SOIL,” and if not, it will show “UNHEALTHY SOIL.” The code also includes different crop options such as eggplant, pechay, ampalaya, bell pepper, and tomato for future improvements of the project. To use the project, the NPK sensor is placed into the soil sample. After that, the Arduino Uno reads the nutrient levels from the soil. The LCD screen will then display the values of Nitrogen, Phosphorus, and Potassium, together with the soil condition whether it is healthy or unhealthy. The system helps farmers, gardeners, and students easily monitor soil nutrients without doing complicated testing.

This project is important because it helps people understand the condition of the soil before planting crops. It can help improve plant growth, avoid wasting fertilizer, and support better farming. Through this project, we also learned more about coding, electronics, sensors, and how Arduino can be used in real-life applications.
